Introdução e exposição do tema do trabalho

Proposta do trabalho

O trabalho proposto aqui consiste em criar um jogo do estilo Plataforma em 2D utilizando a biblioteca pygame da linguagem python.

Oque é um jogo plataforma?

Um jogo plataforma se caracteriza por ter um Personagem principal que é controlado pelo jogador, e esse personagem deve poder se movimentar para direita, esquerda, para cima e para baixo pelo cenário do jogo, além de pular e ter uma simulação do conceito de gravidade que o puxa para baixo quando pular. Essa movimentação é oque possibilita que ele vença a altura e largura das plataformas, seguindo sempre em frente nas diversas Fases. Fases são sequencias de plataformas que devem ser vencidas para que se chegue ao fim da fase.

Entrega trabalho 0 - (04/10/2015)

Apresentação do Trabalho 0

No trabalho zero o grupo se preocupou em aprender e aplicar corretamente as várias funções pré definidas da biblioteca pygame. O foco aqui foi criar uma estrutura que atenda aos requisitos e conceitos de um jogo Plataforma 2D, criando:

  • Personagem principal
  • As plataformas
  • Conceito de colisão entre o personagem principal e as plataformas
  • Conceito de Fase
  • Estrutura que possibilita a construção de várias fases
  • Estrutura que possibilita dar para as fases propriedades próprias como:
    • posicionamento diferente de plataformas
    • fundo del tela da fase
    • Posição de inicialização do personagem principal e de futuros objetos na fase
    • Fazer a tela "andar" por meio de uma câmera simulada que segue sempre o jogador atualizando e redesenhando as plataformas conforme o jogador movimenta o personagem
